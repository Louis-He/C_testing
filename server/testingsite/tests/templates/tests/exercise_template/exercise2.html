Exercise 2 - Mad about time

Estimate time of completion: > 1 hour for part 1 and 30 mins for part 2

Brief Introduction

In this exercise, you will learn how to process input, output, various statements like if and for, and functions using programming languages C.

Instruction

This is the second exercise of this lecture series, the instruction is similar to exercise #1. Please follow the instruction carefully to procceed.

1. Download and decompress the code skeleton of exercise #2 (Please select according to your operating system).
2. Implement your code in sample.c. Please do not edit any other files.
3. After you test your code locally, now it is time to test your code using online tester. Download Input file in Step 2 and store the file with name input.in. Move the input file to the same directory that has the skeleton code that you decompressed in 1.
4.1 If you are a mac user, please enter the directory in terminal, and run sh compile.sh.
4.2 If you are a windows user, please enter the directory in powershell, and run powershell.exe compile.ps1.
5. compile.sh and compile.ps1 will do everything for you and generate an output file output.out.
6. Do not edit output file, submit your input.in and output.out to the online tester in step 3.
MacOS Windows

Input

The input will be two datetime. Each datetime is represented by 6 integers.
2020 7 8 12 4 8 // year 2020, July, 8th, 12:04:08
In a nutshell, for each cases, you will have a total of 12 integer inputs.
Note: The input will be some time later than Jan, 1st, 1970, midnight. However, there is no guarantee that the input is always correct. The only thing you know is that there will be a total of 12 integer inputs.

Part I Outupt

The output should be two int that represents how many seconds the datetime away from Jan, 1st, 1970, midnight.
1234213\n
1321432\n
Note: If the input is invalid, the output should be -1\n.

Part I Example


        Input:
        1970 1 1 0 0 1
        1970 1 1 1 0 0
        Output:
        1
        3600

        Input:
        1970 1 1 0 0 0
        2020 2 30 0 0 0
        Output:
        0
        -1

Part II establishes upon part I. Now instead of output two integers, you should output the time difference between two datetime. The output should be a string. Do not delete the functions you wrote in part I, continue adding more code and complete this part.

Part II Outupt

The output should be a string that represents the time difference between two datetime: "dd Day(s), hh Hour(s), mm Minute(s), ss Second(s)".
01 day, 23 hours, 45 minutes, 01 second\n
Note1: You always need to output at least two digits for days, hours, minutes and seconds. If the value is less than 10, you need to pad zero in front of it. (e.g: 1 days, 2 hours, 3 minutes, 4 seconds -> 01 day, 02 hours, 03 minutes, 04 seconds)
Note2: Be careful about singular and plural form.
Note3: If the input is invalid, the output should be Error: invalid inputs\n.

Part II Example


        Input:
        2020 7 8 1 7 0
        2020 7 8 2 7 1
        Output:
        00 days, 01 hour, 00 minutes, 01 second

        Input:
        1970 1 1 0 0 0
        2020 2 30 0 0 0
        Output:
        Error: invalid inputs // There is no date 30 in Feb
