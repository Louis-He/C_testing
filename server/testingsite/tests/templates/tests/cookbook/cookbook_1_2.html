My first program

You can type the following code to a new source code file or copy & paste. I encourage typing the code word by word. Also you can use onlinegdb (do not forget to choose C/C++14 in language option) for convenience.

  • C
  • C++ 
#include <stdio.h>

int main(){
  printf("Hello World\n");
  return 0;
}
Summary

The program will print "Hello World" to the terminal

More details!

Let's take a minute to understand what we just wrote. It's okay that you can't understand what I will tell you in a minute at this point...

Line #1 #include <stdio.h> is used to include a library that let you to call printf on line#4.

Line #3 & Line #6 are the beginning and the end of the main function of your program. The program will start execution from main().

Line #4 is the only line that actually does something that you can observe. This line will print "Hello World" to the terminal. What does "\n" do? It is the special character means a new line. 

#include <iostream>

int main(){
  std::cout << "Hello World" << std::endl;
  return 0;
}
Summary

The program will print "Hello World" to the terminal

More details!

Let's take a minute to understand what we just wrote. It's okay that you can't understand what I will tell you in a minute at this point...

Line #1 #include <iostream> is used to include a library that let you to call cout on line#4.

Line #3 & Line #6 are the beginning and the end of the main function of your program. The program will start execution from main().

Line #4 is the only line that actually does something that you can observe. This line will print "Hello World" to the terminal. What does "endl" do? It means a new line.
Then what is that strange "std::" before cout and endl?
Here, "std" is a namespace. More details can be found by this link. To be short, it is like a local code.
For example, there are two classrooms in a school, both have a student whose id is #2020. Then we may distinguish them by calling them "Class A - student 2020", and "Class B - student 2020" respectively. Here, "Class A" and "Class B" can be seen as two namespaces. "student 2020" is only meaningful within that namespace. If you want to express these two students in C++, you can write them as Class_A::sutdent_2020 and Class_B::student_2020.
