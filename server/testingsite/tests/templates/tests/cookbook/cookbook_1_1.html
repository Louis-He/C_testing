Before you start, go and open the terminal (dashboard -> utility -> terminal)
You should see a terminal window like the one from movies. There will be input prompt like the following: 
LouisHe@MacBook-Pro ~ %
Now type g++ and enter. 
LouisHe@MacBook-Pro ~ % g++
g++: fatal error: no input files
compilation terminated.
If you see this, then you don't have to do anything else. Otherwise, you need to run the following. The command is used to install the C & C++ compiler. You may need to enter your mac password. Please follow the intructions of the popup window to install it. 
LouisHe@MacBook-Pro ~ % sudo xcode-select install

What did I just do?

You may wonder what I was doing? Why I need to install a compiler(?)? What does that even mean?

Basically, a compiler is a program that converts the code that human writes (human readable but not machine executable) to a binary file that is machine executable (machine code). The binary file is consists of purely 0s & 1s. That is why the programming language we learn today are mostly called high level languages. The code we write cannot be run by machine directly without being "translated" to machine code.
