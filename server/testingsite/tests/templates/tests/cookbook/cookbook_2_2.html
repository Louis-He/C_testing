Compilation

Programs written in C/C++ languages needs to be compiled to a binary file which your host machine can run. In Contrast, there are interpreted languages that doesn't need to be compiled but needs an interpreter at runtime to execute the program. One of such languages that is popular is Python.

Why do we need a compiler?

We as human write programs that are human readable, but they are not machine readable. Computers only understand basic instructions written in 1's and 0's. Therefore, if we want our program to run, we need a helper tool to translate human-readable code (C/C++ code) to machine code (Binary Executable). Compiler is the one who does the job. In C, we use "gcc" as our compiler. In C++, we use "g++" as our compiler.
