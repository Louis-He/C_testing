OOP Introduction

What is OOP? The simple answer is Object Oriented Programming. However, what is Object Oriented Programming? I will say the three parts of OOP are encapsulation, inheritance and polymorphism. You still don't get what I am saying? Don't worry, we prepared a whole unit for the OOP concept. Let's start with some basic concept: class and object.

Class and Object

Class can be interpreted as a template for its objects. Objects are the instance of a class. It is similar to struct that we learned in C (struct also exists in C++). Let's see an example. We have a class called Human. There are many metrics for a person, like gender, height, weight, and etc. Now we have an object of class Human, let's call him Tom. Tom is a man, 180cm tall, and has a weight of 60kg. We have another person (or object), Alice. She is a woman, 170cm tall, and has a weight of 50kg.

Given the above example, we can think class as an abstract template for something, and object as a detailed instance of the corresponding class. More examples like we can make fruit as a class. Banana, apple, and avocado can be some objects of fruit class.

Now, how can we represent these data written in C++? Let's look closer at the Human example. First, review what we learned in struct. If we want to express this using struct the code should be similar to the one below. 

struct Human{
  int gender; // 1: male, 2: female, 3: other
  double height;
  double weight;
}

int main(){
  struct Human Alice, Tom;
  // access data in struct
  Alice.gender = 2;
  Alice.height = 170.0;
  Alice.weight = 50.0;

  Tom.gender = 1;
  Tom.height = 180.0;
  Tom.weight = 60.0;
  return 0; 
}

Similarly, we can express it using class 

class Human{
  int gender; // 1: male, 2: female, 3: other
  double height;
  double weight;
}

int main(){
  Human Alice, Tom;
  /* The remaining code is the same as the above code segment*/
}

You can see that member data in class is nearly identical to the data in struct. Now we can move on to something that you can't see in C, member function.
