Introduction

What actually happens when our programs are running? How these variables we declared and manipulated are stored in our computer? Are they stored in memory? or on disk? When we try to assign a new value to a variable (e.g. a = 0;), how do our program know which place in our computer needs to be updated?

Let's tackle these problems one at a time. First, we need to know that the variables we declared, and everything we manipulate during runtime of our program store in the main memory of our computer or sometimes we call it RAM(Random Access Memory). We are talking about the same thing that you may heard when you bought your computer. Most of the modern computers have RAM over 4GB.

Now, we have a very simple program below. It first declared a variable a, and assign 0 to a. Later, we change the value of a from 0 to 1. Let's run it line by line and see what happends in the memory.

  • C
  • C++ 
#include <stdio.h>

int main(){
  int a = 0;
  a = 1;
  return 0;
}
#include <iostream>

  a = 1;
