Array and Pointers

Now we know how to declare an integer, a boolean, a character and etc. But how are we going to declare let's say a hundred or a thousand integers? There is no *&%^@ way that we need to write 1000 lines of code just to declare these variables. Luckily, we have something called array.

Here is how it gets declared. 

int arr[100]; // array of size 100
int integers[5] = {0, 1, 2, 3, 4}; // array of 5 integers and initialized from 0 ~ 4
int zeros[10] = { 0 }; // all elements will be zero

How are we going to access the specific element in the array or change it? First, let me tell you the index of the array. It is different from our real life that we always start counting from 1. In C++, you start to count from 0. The first element in the array (leftist element) will be the 0-th element, and then from left to right will be 1-st, 2-nd, and etc. So let's say we want to get 2 from array integers, we will write integers[3]. If we want to change it to zero, the statement will be integers[3] = 0;

Here is a piece of code that will set the value of the element to zero from left to right for every two elements starting from the leftist element. 

int main(){
  int integers[5] = {0, 1, 2, 3, 4};
  // count from 0, ends at 4 (0 -> 4: 5 elements)
  for(int i = 0; i < 5; i++){
    // set element to zero every two elements
    if(i % 2 == 0){
      integers[i] = 0;
    }
  }
  return 0;
}

What is an array?

Before we continue, here is a question. What is an array anyway? We use one variable name to represent an array? How is that possible? Don't we only have one memory block for one variable? Well, you are right, we may need multiple memory blocks to store all the data in an array, but we really only need one memory block to access the whole array! How? See the following code and the memory space allocation. 

int main(){
  // we declared an array of size 5. Initializing them from 0 to 4 
  int integers[5] = {0, 1, 2, 3, 4};

  // integers is actually a pointer!
  int *arr_ptr = integers;

  return 0;
}

Surprise, surprise. Array is actually represented by a pointer! Why can we do that? Notice that the address of the memory space allocated for an array is consecutive.

Pass an array into a function

Because we can use a pointer to represent an entire array, therefore if we want to pass an array to a function, we only need to pass in a pointer. Example as follows 

void foo(int* arr, int size){
  if(size >= 1) {arr[0] = 0};
}
What will happen to the code above? If there is any element in the arr, we will set the first element to zero.
